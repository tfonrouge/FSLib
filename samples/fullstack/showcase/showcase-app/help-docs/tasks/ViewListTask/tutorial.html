Task List Tutorial

Welcome to the Task Manager! This view shows all your tasks in a powerful data grid. Let's walk through what you can do here.

Step 1: Browse tasks
The table displays all tasks with columns for ID, Title, Assignee, Priority, Status, and Estimated Hours. Colored badges make it easy to spot critical items at a glance.
Step 2: Sort columns
Click any column header to sort ascending or descending. Click again to toggle the sort direction. You can sort by title, assignee, priority, or any other column.
Step 3: Filter rows
The Title and Assignee columns have a built-in filter input right below the header. Start typing to narrow down the list instantly. Filtering happens server-side, so it works the same way with thousands of rows.
Step 4: Create a new task
Click the New button in the toolbar to open the task form. Fill in the details and save your new task.
Step 5: Edit or delete
Click any row to open it for editing. Use the trash icon column on the right to delete a task directly from the list.
Tip: The grid supports server-side pagination. Use the page controls at the bottom to navigate through larger data sets.
