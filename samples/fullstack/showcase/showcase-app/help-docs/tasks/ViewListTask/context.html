Task List Reference

Columns

ColumnDescription
IDUnique task identifier. Sortable.
TitleTask name. Sortable and filterable.
AssigneePerson responsible. Sortable and filterable.
Priority Critical High Medium Low
StatusCurrent task state: Open, InProgress, Done, or Cancelled.
Est. HoursEstimated effort in hours.
(delete)Click the trash icon to remove a task.

Toolbar Actions

ButtonAction
NewOpens an empty form to create a task.
RefreshReloads the current page of data.

Filtering

Type in the filter box below Title or Assignee to search. Filters use a "contains" match and are processed on the server.

Pagination

Results are paginated. Use the navigation controls at the bottom of the table to move between pages.
