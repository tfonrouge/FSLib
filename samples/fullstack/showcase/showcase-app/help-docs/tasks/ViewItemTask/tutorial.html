Task Form Tutorial

This form lets you create or edit a task. Here's a quick walkthrough of each field.

Step 1: Title (required)
Give your task a clear, descriptive name. This is the only required field — you can't save without it.
Step 2: Assignee
Enter the name of the person responsible. This is optional — unassigned tasks can be claimed later.
Step 3: Priority and Status
Use the dropdowns to set the priority level (Low through Critical) and the current status (Open, InProgress, Done, or Cancelled). Both default to sensible values.
Step 4: Estimated Hours
Enter an estimate of the effort in hours. Use the spinner arrows or type a number directly.
Step 5: Description
Add any extra context, notes, or acceptance criteria. This is a free-text area — go as detailed as you like.
Step 6: Save
Click Save in the toolbar to persist your changes. The form validates required fields before submitting.
Tip: After saving, you'll be redirected back to the task list where you can see your changes reflected immediately.
