Task Form Reference

Fields

FieldTypeRequiredNotes
TitleTextYesThe task name displayed in the list.
AssigneeTextNoWho is responsible for this task.
PriorityDropdownYesLow, Medium, High, or Critical.
StatusDropdownYesOpen, InProgress, Done, or Cancelled.
Estimated HoursSpinnerNoWhole number between 0 and 1000.
DescriptionText areaNoFree-form notes or details.

Toolbar Actions

ButtonAction
SaveValidates and saves the task (create or update).
CancelDiscards changes and returns to the list.
DeleteRemoves the task (only visible when editing an existing task).

Validation

The form won't submit until all required fields are filled in. Invalid fields are highlighted with a red border and an error message appears below.
